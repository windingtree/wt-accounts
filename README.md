## Install
```
python3 -m venv ENV || virtualenv --python=/usr/bin/python3 ENV
source ENV/bin/activate
pip install -r requirements.txt
```

## Usage
```
./manage.py migrate
./manage.py runserver
```

Go to `http://localhost:8000/accounts/login`
